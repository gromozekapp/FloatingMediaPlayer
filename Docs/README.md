# Docs

## Demo GIF

Before making the repository public, add a screen recording:

1. Run the demo in Xcode (iOS Simulator or device).
2. Record the floating player: drag, play/pause, seek via progress ring.
3. Export as GIF (e.g. via [Kap](https://getkap.co/) or `ffmpeg`).
4. Save as `Docs/demo.gif`.
5. Uncomment the demo line in the root `README.md`:

```markdown
![Demo](Docs/demo.gif)
```

Recommended size: ~800×600 px, under 5 MB.
